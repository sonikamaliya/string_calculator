# string_calculator
Calculate numbers from given string

Clone the repo in local and navigate to the project directory.

Install rspec if not installed locally

To Run the rspec test cases:

    rspec ./string_calculator_spec.rb

To run string calculator and iterate with different type of string addition

1. Open irb in current project directory terminal 

2. Then run below steps:

    require "./string_calculator"

    StringCalculator.add("//;\n1;2") # output will be 3 (As in given string includes 1,2 as number)
